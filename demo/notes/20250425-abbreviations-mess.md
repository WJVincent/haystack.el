## pkm landscape right now

So many tools. Trying to keep track:

**Emacs-based:**
- org-roam (sqlite, backlinks, graph viz)
- denote — plain files, silo-free, emacs-native
- haystack — search-first, no db, ripgrep backend
- plain org files + grep (the "no tool" tool)

**Non-emacs:**
- Obsidian — electron app, markdown, plugins ecosystem
- Logseq — outliner, open source, local-first
- Notion — cloud, proprietary, good for teams
- Roam Research — the original "networked thought" app

The thing about personal knowledge mgmt is that the tool matters less
than the habit. People spend more time configuring their PKM system
than actually writing notes. I've done this. Everyone's done this.

Haystack's advantage: minimal config, just write files and search.
No graph database, no sync server, no electron runtime.

The disadvantage: no backlinks, no graph visualization, no mobile app.
But are those features or distractions? Genuinely unsure.

emacs is the common thread for the first group. If you're already in
Emacs you don't need another app. If you're not in Emacs... well,
that's a different conversation.
