And this one references a file that was renamed:
[[file:old-name-that-changed.org][Renamed Note]]

The point of this note is that broken links are a fact of life in any
plain-text knowledge base. Haystack's approach (search, don't link)
sidesteps this problem entirely. You never have a broken link if you
never create links — you just search for the content.
