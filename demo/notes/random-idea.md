# What if notes could link themselves?

Instead of manually linking notes, what if the system could detect
overlapping vocabulary and suggest connections automatically?

This is basically what discoverability does in haystack but it could
go further — rank suggestions by frecency, surface them in a sidebar,
maybe even auto-insert bidirectional links.

Not sure if this is a good idea or if it would just create noise.
Need to think more about the signal/noise tradeoff.
